## 三、申请流程时间规划

| 阶段 | 预计时间 | 主要工作 | 责任人 |
|------|----------|----------|--------|
| 材料准备阶段 | 1-2周 | 填写发明人/申请人信息、准备委托书等 | 申请人 |
| 代理机构委托 | 1-3天 | 选择专利代理机构、签订委托协议 | 申请人 |
| 申请文件撰写阶段 | 2-3周 | 专业代理师撰写正式申请文件 | 代理师 |
| 申请文件审核 | 3-5天 | 双方确认最终申请文件 | 申请人/代理师 |
| 正式提交申请 | 1天 | 向国家知识产权局提交申请 | 代理机构 |
| 受理阶段 | 1-2周 | 等待受理通知书 | - |
| 初步审查阶段 | 3-6个月 | 形式审查，补正等 | - |
| 公布阶段 | 18个月 | （可提前公开） | - |
| 实质审查阶段 | 1-3年 | （需提实审请求） | - |
| 授权或驳回 | - | 根据审查结果 | - |

## 四、代理机构选择建议

### 4.1 代理机构资质要求

- [ ] 国家知识产权局注册专利代理机构
- [ ] 有农业信息化/物联网/人工智能领域代理经验
- [ ] 擅长计算机软件/算法类发明专利申请
- [ ] 有丰富的高价值专利授权案例

### 4.2 代理机构推荐方向

1. 北京地区：优秀专利代理资源集中
2. 上海地区：国际化程度高
3. 本地代理：沟通便利，费用相对较低

## 五、注意事项

### 5.1 提交前检查清单

- [ ] 所有申请文件已按专利局规定格式排版

- [ ] 附图标记与说明书一致
- [ ] 发明人/申请人信息准确无误

- [ ] 确认在申请日前未公开发表相关技术
